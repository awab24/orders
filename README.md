# restaurantapp

Full-stack restaurant orders app using React (Vite), Node/Express, and Supabase.

## Setup
- Frontend: `cd Frontend && npm install && npm run dev`
- Backend: `cd backend && npm install && cp .env.example .env` then fill Supabase keys and run `npm start`
- Environment: provide Supabase project URL and anon key for the frontend; service role key for backend writes.
